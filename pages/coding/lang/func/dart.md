# Language constructs

## Lists

- Expressed using `[]`
- eg `var favorites = [];`
    - this is an empty list
- To specify the type of objects in the list, use [generics]

## Sets

- Expressed using `{}`

## Generics

- Expressed using `<>`
- eg `var favorites = <WordPair>[];`
    - this is an empty list of objects of type `WordPair`
    - (instances of the `WordPair` class)
